# Hermes Dev Orchestra

> **Status:** v1.2 migration in progress (2026-04-28)

## What This Is

Hermes Dev Orchestra is an adapter layer that orchestrates Claude Code (supervision) and Codex CLI (execution) for multi-project AI-assisted development, with explicit risk escalation and human-in-the-loop for L3/L4 decisions.
