# Phase 8 Plan 01: CLI Shell, Packaging & Command Envelope

## Objective
- **What:** Build the runnable `hermes` CLI shell with help/version and structured JSON result envelopes.
- **Why:** Phase 9–12 commands need a stable entrypoint and output contract before adding durable behavior.
- **Output:** Local no-sudo Node CLI with explicit command dispatch and envelope helpers.

## Threat Model

<threat_model>
Phase 8 introduces a local executable. Risks are limited to misleading command output and accidental dependency or privilege requirements. Mitigations: no external dependencies, no shell execution, no sudo requirement, explicit exit codes, and structured errors that do not leak secrets.
</threat_model>

## Verification
- Run `node bin/hermes.js --help`.
- Run `node bin/hermes.js --version`.
- Run `node bin/hermes.js unknown --json` and validate JSON fields.
